# Portfolio Repository: Go & React Learning Journey

This repository is a personal portfolio that documents my journey in full-stack development using Go and React. It includes foundational exercises, mini projects, and a more complete application built from scratch.

## What’s Included

- Go basics and core programming concepts
- JavaScript and React fundamentals
- A full-stack project management app named Sproad

## Projects

### 1. Go Basics

A collection of practice files covering:

- variables and data types
- functions and pointers
- structs and maps
- loops, conditions, and error handling
- JWT and basic API concepts

### 2. JavaScript & React Basics

A set of beginner-friendly front-end exercises using:

- HTML, CSS, and JavaScript
- React components and state
- simple UI implementations

### 3. Sproad

A full-stack project management application inspired by Trello, featuring:

- authentication and authorization
- board, list, and card management
- collaboration features for project workflows
- backend API with PostgreSQL
- React frontend with Vite

## Tech Stack

- Backend: Go, Fiber, GORM, PostgreSQL
- Frontend: React, Vite, JavaScript/JSX
- Dev Tools: Docker, Docker Compose

## Project Structure

- golang-basic/ — Go learning exercises
- js_react_dasar/ — React and JavaScript practice projects
- Sproad/ — full-stack project management application
  - back_end_server/ — Go API backend
  - front_end_client/ — React frontend

## Getting Started

### Backend

```bash
cd Sproad/back_end_server
go mod download
go run main.go
```

### Frontend

```bash
cd Sproad/front_end_client
npm install
npm run dev
```

### Full Stack with Docker

```bash
cd Sproad
docker compose up --build
```
